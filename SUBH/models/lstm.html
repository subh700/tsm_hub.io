LSTM Model

Long Short-Term Memory Network for Time Series Prediction

Overview

Long Short-Term Memory (LSTM) networks are a specialized type of Recurrent Neural Network designed to learn long-term dependencies. They were introduced by Hochreiter & Schmidhuber (1997) and were refined and popularized by many people in following work.

LSTMs are explicitly designed to avoid the long-term dependency problem. Remembering information for long periods of time is practically their default behavior, not something they struggle to learn!

Architecture

The key to LSTMs is the cell state, which runs straight down the entire chain, with only minor linear interactions. Information can easily flow along it unchanged. The LSTM can remove or add information to the cell state, carefully regulated by structures called gates.

Implementation

Here's a basic implementation using PyTorch:


import torch
import torch.nn as nn

class LSTMModel(nn.Module):
    def __init__(self, input_size, hidden_size, num_layers):
        super(LSTMModel, self).__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers, batch_first=True)
        self.fc = nn.Linear(hidden_size, 1)
    
    def forward(self, x):
        h0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size)
        c0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size)
        out, _ = self.lstm(x, (h0, c0))
        out = self.fc(out[:, -1, :])
        return out
